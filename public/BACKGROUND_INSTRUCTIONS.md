# Background Image Setup

To use your background image:

1. Save your image file as `background.jpg` in this folder (`/public/`)
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
3. The image will automatically be used as the website background

**Note:** The background image will be:
- Set to cover the entire viewport
- Fixed in position (parallax effect)
- Overlaid with a semi-transparent grey layer for content readability

If you use a different filename, update the CSS files:
- `src/styles/studentHome.css` - Change `url('/background.jpg')` to your filename
- `src/styles/adminDashboard.css` - Change `url('/background.jpg')` to your filename
